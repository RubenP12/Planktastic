Export Site

Download all pages with your content baked in, ready to upload to any web server.

1

Save your changes

Click Save Changes in the top right to make sure all edits are included in the export.

2

Click Export

A single file called content-data.js will download. It contains all your content — text, news, team members, images and links.

3

Upload one file

Upload content-data.js to your server inside the js/ folder, replacing the existing file. That's it — no server required, no ZIP to unpack.

First time uploading? You need to upload the full project folder once (all HTML files, css/, js/, logos/). After that, only content-data.js needs to be re-uploaded each time you make changes.
